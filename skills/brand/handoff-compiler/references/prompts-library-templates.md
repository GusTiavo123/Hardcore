## Universal Prompt Structure

Every prompt follows this block format:

```markdown
### {PROMPT_NAME}

**Goal**: {what to build}

**Layout**: {sections, components, hierarchy}

**Content**:
- {specific content item 1 — from verbal output when applicable}
- {specific content item 2}
- {...}

**Audience**: {from strategy.target_audience_refined}

**Voice**: Write copy that is {VOICE_ATTRIBUTES}. Reference do/don'ts from brand book:
- Do: {DO_EXAMPLE}
- Don't: {DONT_EXAMPLE}

**Design**: Apply published design system. {specific design notes if any}

**Before building**: Propose 4 distinct visual directions tailored to this brief (each as: bg hex / accent hex / approach — one-line rationale), then ask me to pick one and implement only that direction.

---
```

The "Before building" move is critical — it forces Claude Design to propose options before committing to a direction.
